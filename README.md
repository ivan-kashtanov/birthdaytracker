**Веб приложение Поздравлятор**

Данное приложение реализует функцию ведения записей о днях рождений. Приложение является CRUD системой. Также в 
в приложении реализована рассылка напоминаний по почте mail автоматически в заданное время в конфигурации проекта.
Для рассылки необходимо указать почту с которой будет рассылка, а также специальный пароль для разработки.

Внутри данного проекта реализован также фронт, который находиться в ресурсах в статике.

В приложении подключен Swagger для документации, бэк реализован на Spring Boot. База данных - Postgres. Сборка проекта - Gradle

Для развертывания приложения используется Docker. Для запуска приложения необходимо собрать образ, внутри одного контейнера
находятся как само приложение так и база данных.
